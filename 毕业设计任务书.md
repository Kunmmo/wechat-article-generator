## （一）课题来源、意义与主要内容

> 注明自拟、科研、科技服务类别及任务提出单位

### 课题来源

自拟。项目背景源自 [深灵.Tech](https://www.deepling.tech/) —— 一个由 AI 自主运营的技术自媒体平台。

### 课题意义

随着微信公众号成为中文互联网最重要的内容分发渠道之一，高质量图文内容的创作需求日益增长。然而，现有大语言模型（LLM）在内容创作中普遍存在以下痛点：

1. **内容平庸化**：单一模型输出往往陷入"平庸正确"的困境，缺乏深度洞察和独特观点
2. **缺乏网感**：生成内容语言规范但缺少情感共鸣，难以引发读者的阅读兴趣和传播欲望
3. **配图生硬**：图文搭配缺乏语义关联，表情包和配图选择不当影响阅读体验

因此，如何利用多模态大模型的跨模态理解能力，结合多智能体协作机制，实现高质量、有深度、具有传播性的微信公众号图文内容自动生成，是当前亟待解决的技术难题。本研究将为 AI 辅助内容创作领域提供新的方法论和技术框架。

### 主要内容

本研究提出一种"多智能体竞争与协作环境"（Multi-Agent Debate/Competition Environment）架构，通过三个具有不同"人格"的智能体之间的博弈与融合，产出高质量图文内容。同时，基于跨模态检索技术实现表情包和配图的智能适配。

**本研究的主要目标是：**

1. **设计三智能体人格架构**：构建深度思考者（负责内容深度）、Meme大师（负责网感与传播性）、铁面主编（负责仲裁与图文对齐）三个智能体，分别基于 DeepSeek、GPT-4o、Gemini 实现。
2. **构建多轮辩论工作流**：实现"草稿生成 → 毒舌点评 → 融合定稿"的迭代优化流程，通过智能体间的博弈产出高质量内容。
3. **实现智能图文适配系统**：基于 CLIP 构建表情包向量库实现语义检索，结合 Gemini/Stable Diffusion 生成配图，使用 CLIP Score 评估图文对齐质量。
4. **整合数据源与输出渲染**：接入 WeRSS 数据采集系统，实现 HTML/Markdown 图文预览导出。

## （二）目的要求和主要技术指标

### 目的要求

设计并实现一个基于多智能体辩论机制的微信公众号图文内容生成系统，通过智能体间的竞争与协作，打破 LLM 输出"平庸正确"的局限，生成既有深度又有网感的高质量图文内容。同时，利用跨模态检索和生成技术，实现表情包和配图的智能适配，提升图文一致性和阅读体验。

### 主要技术指标

1. **三智能体架构**：实现三个不同人格的智能体，支持多轮辩论工作流
2. **内容生成**：生成 2000-4000 字图文内容，自动插入 3-6 个表情包/配图标记
3. **图文对齐**：基于 CLIP 语义检索表情包，支持 Gemini/SD 生成配图
4. **系统功能**：提供 CLI 和 Web 界面，支持 HTML/Markdown 输出
